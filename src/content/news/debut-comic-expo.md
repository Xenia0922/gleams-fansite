# 🎉 Gleams 出道首演！

星界的公主们，终于要与大家见面了！

2026年1月31日，Gleams 将在 **南宁·第一届 Comic Expo 国风动漫展** 完成团体首次登台演出。三位公主——💛白菜、💙云团、💚柚子——将身着公式服1.0，带来精心准备的舞台！

## 公演详情

| 项目 | 内容 |
|------|------|
| 日期 | 2026年1月31日（六） |
| 时间 | 12:40～13:00 |
| 地点 | 南宁北投明月荟（西乡塘区） |
| 服装 | 公式服1.0 |
| 出演 | 白菜、云团、柚子 |

## 世界观

> 在星界尽头，存在着一座神秘王国，由水晶支撑着整个王国的运转。有天能蚕食人间星光的阴霾突然降临，王国赖以生存的水晶随之日渐暗淡无光。正是在此存亡之际，来自不同城堡的公主，在命运的指引下相聚于此，公主们开始踏上寻找名为「Gleams」的能量宝石来守护他们的世界。

**南宁的骑士们，准备好了吗？我们现场见！** ⭐
