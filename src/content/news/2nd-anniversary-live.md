# Gleams 2周年記念ライブ「Starlight Symphony」開催決定！

いつも Gleams を応援していただき、ありがとうございます。

2026年7月20日（日）、Gleams が結成2周年を迎えるにあたり、渋谷 WWW X にて記念ライブ **「Starlight Symphony」** を開催することが決定しました！

## 公演詳細

| 項目 | 内容 |
|------|------|
| 日時 | 2026年7月20日（日）開場 17:30 / 開演 18:30 |
| 会場 | 渋谷 WWW X |
| チケット | 前売 ¥4,500 / 当日 ¥5,000（+1ドリンク代） |
| 出演 | Gleams 全員（Rena、Yuna、Miku、Koharu、Aoi、Rin） |
